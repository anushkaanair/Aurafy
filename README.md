# 🚀 Aurafy

### Design Your AI Stack in Seconds

Aurafy is an **AI architecture design platform** that helps developers plan and design AI-powered products.

Instead of spending hours researching which models, infrastructure, and pipelines to use, Aurafy analyzes your product idea and generates a **complete AI system architecture**.

Aurafy acts like an **AI system architect** for developers building modern AI products.

---

# 🌐 Aurafy Web Platform

Aurafy is designed as a **web-based developer tool**.

Users can open the platform, describe their idea, and instantly receive:

• AI tasks required
• recommended AI models
• system architecture pipeline
• estimated operational cost
• complexity analysis
• performance simulation

This allows developers to **plan AI infrastructure before writing production code**.

---

# ✨ Platform Features

### 🧠 AI Idea Analyzer

Users describe their product idea and Aurafy detects the required AI capabilities automatically.

Examples:

* chatbot
* image generation
* speech recognition
* embeddings
* video generation

---

### 🤖 Model Recommendation Engine

Aurafy selects the most suitable models for each AI task.

Example models:

* GPT-4o-mini
* Claude-3-Haiku
* Stable Diffusion XL

The system ranks models based on **task suitability and cost efficiency**.

---

### 🏗 AI Architecture Generator

Aurafy generates a production-style AI architecture pipeline including:

* API gateways
* AI model services
* storage layers
* frontend integration

---

### 💰 Cost Estimation

Aurafy estimates the cost per request based on selected AI models.

This helps developers plan **scalable AI systems with predictable costs**.

---

### 📊 Complexity Analysis

Aurafy evaluates system complexity using signals such as:

• number of AI tasks
• model orchestration
• infrastructure layers
• pipeline depth

The result is a **complexity score for the system**.

---

### ⚡ Performance Simulation

Aurafy simulates how the system would perform in production.

Metrics include:

* latency
* throughput
* scalability risk

# ▶️ Running Locally

Clone the repository

```id="aqdzt0"
git clone https://github.com/anushkaanair/aurafy.git
```

Navigate to the project

```id="smh3bf"
cd aurafy
```

Create virtual environment

```id="hu0k6s"
python -m venv venv
```

Activate environment

Windows

```id="g7lyc9"
venv\Scripts\activate
```

Install dependencies

```id="aq0j4f"
pip install -r requirements.txt
```

Run the backend

```id="wzkbq2"
uvicorn backend.main:app --reload
```

Run the AI strategy engine

```id="qg8ng1"
python run_strategy.py
```
